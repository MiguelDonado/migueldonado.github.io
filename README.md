# My Blog (migueldonado.github.io)

Personal blog built with Jekyll.

## Run locally
```py
bundle install
bundle exec jekyll serve
```

## Create a post
Add a file to _posts/
Format: YYYY-MM-DD-title.md

## Structure
- _data/ → social media 
- _posts/ → blog posts
- _tabs/ → tabs (about, favorite_talks...)
- assets/ → images, css, js
- _config.yml → site configuration
- _site/ → Automatically generated by Jekyll when you build or serve your site
